Coords est une ressource simple et pratique pour FiveM permettant d’afficher et copier facilement les coordonnées du joueur dans plusieurs formats : vector3, vector4, x, y, z, table Lua et heading.

Description complète

Coords est un outil de développement conçu pour les serveurs FiveM, idéal pour récupérer rapidement la position exacte d’un joueur en jeu.
L’interface permet de copier les coordonnées en un clic dans plusieurs formats utiles pour le scripting, le mapping ou la configuration de zones.

Fonctionnalités
Affichage des coordonnées du joueur en temps réel
Copie rapide en un clic
Plusieurs formats disponibles :
vector3(x, y, z)
vector4(x, y, z, h)
x, y, z
{x = x, y = y, z = z}
heading
Interface moderne et lisible
Pratique pour le développement de scripts, PNJ, zones, véhicules, téléportations, etc.
Utilisation

Cette ressource est particulièrement utile pour :

placer des PNJ
configurer des blips
définir des positions de téléportation
créer des zones interactives
récupérer rapidement des coordonnées pour un script ESX/QBCore/Standalone
